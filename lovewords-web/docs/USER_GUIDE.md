## Getting Started

### What is LoveWords?

LoveWords is an AAC (Augmentative and Alternative Communication) web application designed to help you express love, affection, and everyday communication needs through visual communication boards.

### Accessing LoveWords

Visit: **https://lovewords-web.vercel.app/**

The app works on:
- Desktop browsers (Chrome, Firefox, Safari, Edge)
- Tablets (iPad, Android tablets)
- Mobile phones (iOS, Android)

### First Time Setup

1. When you first open LoveWords, you'll see the home board: **Love & Affection**
2. The board displays colorful buttons with words and phrases
3. Tap or click any button to hear it spoken aloud
4. Use the navigation buttons to explore other boards

## Building Messages

### Two Types of Buttons

**Immediate Speech Buttons** (Love & Affection, Feelings, Activities, Questions, Time)
- Speak the phrase immediately when tapped
- Example: Tapping "I love you" speaks "I love you"

**Word Building Buttons** (Core Words, Basic Needs, People)
- Add words to the message bar
- Build sentences before speaking
- Example: Tap "I" + "want" + "help" = "I want help"

## Keyboard Shortcuts

LoveWords is fully accessible via keyboard!

### Navigation

| Shortcut | Action |
|----------|--------|
| **Arrow Keys** | Move between cells on the board |
| **Enter** or **Space** | Activate the focused cell |
| **Tab** (from page top) | Skip to main communication board |
| **Escape** | Close the Settings modal |

### Visual Focus

- A **blue ring** appears around the focused cell
- Use arrow keys to move the focus

## Accessibility Features

### For Screen Reader Users

LoveWords announces:
- Cell labels when navigating
- Board changes ("Navigated to Feelings board")
- Message updates ("Added love to message")
- Speaking status ("Speaking: I love you")

**Recommended Screen Readers:**
- **macOS**: VoiceOver (Cmd+F5)
- **Windows**: NVDA or JAWS
- **Linux**: Orca

### For Keyboard-Only Users

- Full keyboard navigation with arrow keys
- Visible focus indicators
- Skip link to bypass navigation
- Focus trap in Settings modal (Tab cycles through controls)

### For Low Vision Users

- High contrast colors on buttons
- Large, readable text
- Customizable text size (Settings)
- Support for browser zoom (200%+)

### Speech Settings

**Voice**
- Select from available voices on your device
- Different languages and accents available
- Default voice uses system settings

## Tips & Tricks

### Quick Expression

For immediate expressions, use themed boards:
- **Love & Affection**: "I love you", "Kiss me", "Let's cuddle"
- **Feelings**: "I feel happy", "I feel tired"
- **Activities**: "Let's watch", "I want to rest"

### Complex Sentences

For flexible communication, use word-building boards:
- Start with **Core Words** for versatile vocabulary
- Add words from **Basic Needs** for requests
- Include **People** for pronouns and relationships

### Favorite Phrases

Create your favorite phrases using Core Words:
- "I" + "love" + "you" = "I love you"
- "I" + "want" + "to" + "sleep" = "I want to sleep"

### Switching Contexts

- Click "Back" to return to the previous board

## Troubleshooting

### Speech Not Working

**Check these settings:**
1. Device volume is turned up
2. Browser has permission to use audio
3. Speech synthesis is supported (works in all modern browsers)
4. Try a different voice in Settings

**Browser compatibility:**
- âœ… Chrome/Edge (best support)
- âœ… Safari (good support)
- âœ… Firefox (good support)

### Buttons Not Responding

**Try these steps:**
1. Refresh the page
2. Clear browser cache
3. Try a different browser
4. Check if JavaScript is enabled

### Settings Not Saving

Settings are saved to localStorage:
- Works within the same browser on the same device
- Private/Incognito mode may not save settings
- Clearing browser data resets settings
